# Task App

Vienkārša uzdevumu saraksta web aplikācija.

## Funkcijas
- Pievienot uzdevumu
- Apskatīt uzdevumus
- Dzēst uzdevumus

## Tehnoloģijas
- Frontend: HTML, JavaScript
- Backend: Flask (Python)
- Datubāze: SQLite

## Kā palaist

1. Backend:
cd backend
python models.py
python app.py

2. Frontend:
Atver frontend/index.html pārlūkā
